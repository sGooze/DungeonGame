# Управление

стрелки, NumPad - перемещение и взаимодействие с миром

F10 - закрыть окно игры

## Инвентарь

Окно инвентаря разделено на две части: внизу находится инвентарь, а сверху - экипированные предметы.
Каждый экипированный предмет меняет характеристики своего носителя.

Чтобы надеть/снять предмет, нужно щёлкнуть по нему и нажать на кнопку "Использовать".
Некоторые предметы являются одноразовыми и исчезают сразу после использования.

Голод
===

Каждое действие игрока увеличивает его уровень голода. Чем больше уровень голода, тем медленнее восстанавливается 
здоровье. Если голод достигает максимума, то игра заканчивается.

На скорость роста голода также влияет количество экипированных в данный момент предметов.

Чтобы уменьшить уровень голода, необходимо есть пищу, найденную в подземелье.


Известные баги
===

• Иногда игрок может оказаться на уровне без лестниц. Если это произошло, то игру необходимо начать заново.
